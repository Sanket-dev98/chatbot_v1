# 🤖 Tony - AI Chatbot

Tony is a simple AI chatbot built with Python and the Groq API.

## Features
- 💬 Interactive terminal chat
- 🧠 Conversation history
- 💾 Save & load chats using JSON
- ⚡ Fast responses with Groq API
- 🔐 Secure API key using `.env`

## Tech Stack
- Python
- Groq API
- JSON
- python-dotenv

## Run
```bash
pip install -r requirements.txt
python main.py
```

A beginner-friendly AI chatbot project created while learning Generative AI.
